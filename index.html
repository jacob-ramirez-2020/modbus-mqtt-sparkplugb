Modbus to MQTT SparkplugB Gateway

A Python-based edge application that reads Modbus devices (RTU/TCP) and publishes their data to an MQTT Broker using the SparkplugB protocol. Built for rugged remote deployments with deadband filtering, buffering on disconnect, and web-based configuration coming soon.

View Project on GitHub

Key Features

  • ✅ Supports Modbus RTU (RS485) and Modbus TCP (Ethernet)
  • ✅ Publishes SparkplugB-compliant MQTT payloads
  • ✅ Web UI (in development) for dynamic configuration
  • ✅ Deadband filtering to minimize network traffic
  • ✅ Buffering with SQLite database on connection loss
  • ✅ Automatic historical replay after reconnection
  • ✅ API-based logging and runtime status configuration
